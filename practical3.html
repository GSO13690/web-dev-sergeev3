How to open the dev console in Safari/Mozilla

  • Safari: Enable the Develop menu in Safari preferences, then go to Develop > Show JavaScript Console.
  • Mozilla: Press Ctrl + Shift + J

Absolute vs. Relative Links

  • Absolute Links: URLs that direct to a specific, unchanging web address.
  • Relative Links:Hyperlinks pointing to pages within same website.

Use Cases:

  • Absolute Links: Different domain resource linking.
  • Relative Links: Inner navigation within the same site.

Image Formats in Web Development

  • JPG (Joint Photographic Group): Good for photos. Uses lossy compression, reducing file size but may lose quality.
  • GIF (Graphics Interchange Format): Best for simple graphics and animations with few colors. Supports transparency.
  • PNG (Portable Network Graphics): Lossless compression. Great for images with transparency and sharp edges.
  • WebP: Googleâ€™s format for the web. Offers both lossy and lossless compression. Supports transparency and animation.
  • SVG (Scalable Vector Graphics): Vector format. Ideal for scalable graphics like icons and diagrams without quality loss.
